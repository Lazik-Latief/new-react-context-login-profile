# React Context Login Profile

A simple and clean **React login & profile application** built to understand and practice **Context API** and global state management.

This project demonstrates how user data can be shared across components without prop drilling, while also handling **multiple users** using arrays.

---

##  Features

- Login form with username and password
- Global state management using **React Context API**
- Stores multiple users in a shared state
- Displays all logged-in users in the profile section
- Clean and modern UI
- Beginner-friendly and well-structured codebase

---

##  What I Learned

- How Context API works in real applications
- Creating and using a Context Provider
- Sharing state between components using `useContext`
- Managing multiple values using arrays in React
- Separating concerns between components (Login & Profile)
- Writing cleaner and reusable React code
